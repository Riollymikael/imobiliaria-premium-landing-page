## 📋 Sobre o Projeto

O **Luxe Haven Real Estate** é um site institucional e vitrine digital voltada para o mercado imobiliário de luxo (High-End Real Estate). O projeto simula uma imobiliária boutique focada em propriedades exclusivas, oferecendo aos clientes uma navegação fluida, elegante e adaptada para todos os tipos de dispositivos (Desktop, Tablet e Mobile).

---

## ✨ Principais Funcionalidades

* **Página Inicial Dinâmica:** Seção Hero imersiva com formulário rápido de busca por localização e tipo de imóvel.
* **Portfólio em Destaque:** Cards interativos detalhando mansões, coberturas e residências de altíssimo padrão.
* **Página de Detalhes Dedicada:** Sistema dinâmico via URL params (`detalhes.html?id=X`) que puxa informações completas, especificações técnicas detalhadas e descrições aprofundadas de cada imóvel.
* **Seção Sobre a Imobiliária:** Apresentação institucional com métricas de mercado e credibilidade.
* **Depoimentos de Clientes:** Prova social integrada ao layout.
* **Formulário de Contato Exclusivo:** Canais diretos para captação de leads qualificados.
* **100% Responsivo:** Otimização completa para telas menores, garantindo que a visualização de imagens e textos se adapte perfeitamente em smartphones.
